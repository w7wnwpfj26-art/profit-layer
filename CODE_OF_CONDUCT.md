# 行为准则

本项目致力于创建一个开放、尊重、专业且包容的协作环境。参与本项目即表示你同意遵守以下行为准则。

## 1. 我们的承诺
我们承诺为所有参与者提供无骚扰的协作环境，无论其年龄、性别、取向、种族、宗教、国籍或经验水平。

## 2. 允许的行为
- 尊重他人的观点、时间与努力
- 友好、建设性地沟通与反馈
- 关注问题本身，避免人身攻击

## 3. 不可接受的行为
- 侮辱、歧视或骚扰性言论
- 人身攻击、恐吓或恶意行为
- 未经允许发布他人隐私信息

## 4. 执行与处理
若发生不当行为，维护者有权采取包括警告、删除内容、临时或永久禁止参与等措施。

## 5. 报告渠道
如需报告行为问题，请通过项目维护者私下渠道处理。
